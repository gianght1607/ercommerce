Đào tạo Tester theo hướng thực chiến

Lộ trình bài bản, học đến đâu làm được đến đó — đúng quy trình doanh nghiệp thực tế.

Học thực chiến 100%

Mỗi buổi học gắn với bài tập và dự án thực tế, không lan man lý thuyết.

Nhiều hình thức học

Video record, lớp nhóm 2-3 người hoặc dạy kèm 1-1 tùy nhu cầu của bạn.

Hỗ trợ trọn khóa

Tài liệu, bài tập và giải đáp xuyên suốt — không học một mình.

Sẵn sàng đi làm

Ôn tập phỏng vấn, làm CV theo từng level từ Fresher đến Senior.

Kiểm thử phần mềm Manual

  • Nền tảng kiểm thử, SDLC & STLC
  • Phân tích & viết testcase thực tế
  • Agile/Scrum, log bug với Jira
  • SQL, API & chuẩn bị đi làm

API Testing 🔍 Xem chi tiết

Kiểm thử API – Postman & RESTful

Kỹ năng bắt buộc với mọi QA hiện đại: phát hiện lỗi sớm, độc lập với giao diện.

  • Nền tảng RESTful API, HTTP, Status Code
  • Phân tích & viết testcase API
  • Kiểm thử tích hợp UI – Backend
  • Postman thực chiến, data-driven test

SQL 🔍 Xem chi tiết

SQL dành cho Tester

Một QA biết SQL là một QA khác biệt — kiểm tra dữ liệu trực tiếp trong database.

  • Nền tảng CSDL, cài đặt SQL Server
  • SELECT, INSERT, UPDATE, DELETE, JOIN
  • Thực hành với CSDL thực tế
  • Ứng dụng SQL vào công việc QA
